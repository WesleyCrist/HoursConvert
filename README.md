# Time converter

- Convert minutes to hours and hours to minutes

## Conversion examples

- .run('1439') -> '23:59'
- .run('23:59') -> 1439

- Code for now limited to the values shown above

## Examples with code

```

import Convert from "hoursconvert";

const convert = new Convert()

console.log(convert.run('12:12', 732))


```
